Estimador de Aluguéis em Campo Grande/MS

Na disciplina Projeto Integrador III do Curso de Ciência de Dados criei um estimador do preço dos aluguéis em Campo Grande/MS que envolveu desde captação de dados públicos até o deploy utilizando Flask

GitHub Link

Printscreen do estimador de aluguéis

Este projeto exigiu uma abordagem completa do ciclo de desenvolvimento em Ciência de Dados: extração de dados públicos na web, limpeza e tratamento, desenvolvimento de um modelo de machine learning e deploy da ferramenta de forma gratuita.

O projeto também abordou aspectos sociais dos dados levantados sobre os aluguéis em Campo Grande, bem como mostrou uma aplicação de negócio possível para o futuro.
