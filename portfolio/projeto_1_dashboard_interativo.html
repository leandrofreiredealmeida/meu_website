Dashboard Interativo de Xadrez

Na disciplina Projeto Integrador II do curso de Ciência dos Dados desenvolvi um dashboard percorrendo todo o ciclo de desenvolvimento de um projeto da área: captação, limpeza, modelagem, verificação, validação, teste e deploy

Distribuição de xeque-mates por quantidade de lances

Este projeto desenvolveu um dashboard interativo para análise de desempenho no xadrez, utilizando dados das minhas partidas no website Lichess. O objetivo geral é identificar padrões nos meus jogos e nos de meus oponentes, permitindo a melhoria contínua das estratégias de jogo e a visualização de dados detalhados sobre vitórias, derrotas, aberturas utilizadas e erros comuns. A metodologia envolveu a coleta de dados via API do Lichess, seguida pelo pré-processamento e análise exploratória utilizando Python (com bibliotecas como Pandas e Plotly). O dashboard foi desenvolvido com a ferramenta Streamlit, proporcionando uma interface intuitiva e interativa para a exploração dos dados.

Os principais problemas abordados incluem a análise do meu desempenho geral, das aberturas que utilizo e das táticas e estratégias que posso empregar. Adicionalmente, o projeto busca entender como o desempenho varia ao longo do tempo e em diferentes formatos de partida.

O resultado esperado inclui um dashboard funcional e acessível, que não apenas fornece insights sobre o meu desempenho, mas que também serve como uma ferramenta educativa e de análise para a comunidade.
